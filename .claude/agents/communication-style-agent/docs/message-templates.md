# Message Templates (14 templates)

## Post-Call Recap (high-value prospect)
```
Subject: Call Recap

Here's a quick recap of where we landed:

**[Their Businesses & Needs]**
- [Business 1] -- [what they need]
- [Business 2] -- [what they need]

**Pricing Discussion**
[Standard structure: 20% up to $20K, 15% $20K-$40K, 10% above $40K, $1K/account minimum below $5K spend]

**Key Concerns You Raised**
- [Concern 1] -- [how we addressed it]

**Next Steps**
1. [Their action + timeline]
2. [Our action + timeline]
3. [Follow-up call scheduling]

-- Peterson
Creekside Marketing Pros
```

[calendar link]

## Warm Lead from Event/LinkedIn
```
Hey [Name], [1-sentence reference to how they met]. See if you're interested in setting up a call to talk a little bit more about your business and potentially doing some paid media to help [them/their brands] grow. You available any time later this week?
```

## Upwork Initial Pitch (cold, problem-first)
```
[No greeting. Open with problem diagnosis for their industry/situation.]

[1-2 sentences anchoring credibility: "$20M+ managed", "200+ accounts audited"]

[Relevant case study or specific improvement metric]

[Calendar link or "happy to hop on a call to discuss"]
```

## Follow-Up on Stalled Conversation
```
Hey [Name], just wanted to circle back on this -- [1-sentence reminder]. [1-sentence value prop or question]. Let me know!
```
For stalled Upwork after multiple follow-ups: `Have you given up on this project?`

## Internal Delegation (ClickUp/GChat)
```
@[Name] [lowercase instruction]. [Constraint or deadline if applicable]
```
No period at end. No greeting. No sign-off.

## Internal Correction
```
[State what happened factually -- use specific example]
[State what should have happened -- give the principle]
[Explain WHY so they can generalize]
[Close with: "I know this is a really detailed explanation but I would like you to tell me whether or not this makes sense or if I need to clarify on any piece"]
```

## Client Accountability (when Creekside made a mistake)
```
[Direct acknowledgment -- "I won't sugarcoat any of the details"]
[Explain what happened and why -- numbered if multiple issues]
[Take explicit responsibility -- "Sloppy on our end" / "process issue on my part"]
[State what's being done to fix it]
[Pivot to forward-looking action]
```
Key: ONE apology, not repeated. Own it, explain it, fix it, move on. Uses "TL;DR:" at the end if the message was long.

## Client Onboarding (template email)
```
Hi [First Name],

We're excited to kick things off and start building your campaigns! To keep everything streamlined, here's what to expect over the next few days --

1. Join Your Google Chat Channel
2. Access Your Google Drive Folder
3. Timeline Expectations
4. Questions During Onboarding?
```
Note: This is the ONE context where Peterson uses emoji bullet markers (checkmarks, folders, etc.) -- onboarding templates only.

## Training/Coaching Feedback
```
Notes on call:
- [specific correction -- blunt, not softened]
- [specific correction with replacement language]
- [behavioral instruction with threshold: "these calls should be 45min, not 20min"]

[Encouragement embedded in the correction, not separate]
[Optional "lol" after a particularly harsh truth to remove sting]
```
