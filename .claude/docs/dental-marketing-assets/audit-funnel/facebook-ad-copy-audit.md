# Facebook Ad Copy: Dental Practice Offer (Audit Funnel) -- V3
## For Aamir / Cade

**V3 (2026-07-13):** Added the VIDEO AD COPY section: six final headline + primary text sets built and approved by Peterson, paired with the five v2 talking-head scripts (plus one second-opinion set with no video). Descriptions are AI-generated, all within Meta's 30-char description limit; headlines all within the 40-char limit.

**V2 (2026-07-10):** Launch set cut from 13 ads to 5, each paired 1:1 with a v2 static creative (see static-ad-creative-briefs-audit.md). Changes driven by 90-day account data: guarantee/hard-offer structure leads (mirrors the account's cheapest converter), proof copy reframed from "what's wrong with your ads" to "how we'd do the same for your practice," emotional angle kept and funded (highest CTR signal, never tested past $63). **Funnel Role:** Ads catch attention. The prospect's mindset after seeing the ad should be "these people look like they could help me." The landing page convinces. The audit delivers value and converts. Ads do NOT need to tell the full story -- they hook and get the click.

**Specs:** Primary text ~125 chars visible before "See More." Headline 27-40 chars. No em dashes.
